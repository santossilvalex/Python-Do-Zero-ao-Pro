# 🐍 Curso de Python - TéoMeWhy

Este repositório contém meus estudos, anotações e exercícios realizados durante o curso de Python do **TéoMeWhy**. O objetivo é documentar minha evolução técnica, partindo dos fundamentos até tópicos avançados de manipulação de dados e integração com serviços externos.

---

## 📚 Ementa do Curso

### 1. Fundamentos e Estruturas de Controle
*   **Introdução ao Python:** Configuração de ambiente e o clássico "Olá mundo!".
*   **Tipos Básicos de Dados:** Manipulação de Textos (Strings), Números e operações matemáticas.
*   **Variáveis:** Como armazenar resultados e gerenciar entradas do usuário.
*   **Estruturas de Controle:**
    *   Condicionais (`if`, `else`, `elif`).
    *   Repetição (`while`, `for`, `break`, `continue`).
    *   Tratamento de Exceções (`try`, `except`).

### 2. Estruturas de Dados
*   **Listas e Tuplas:** Operações de criação, acesso, modificação e funções integradas (`len`, `append`, `remove`, `sort`).
*   **Dicionários:** Trabalhando com estruturas de chave-valor.

### 3. Modularização e Manipulação de Dados
*   **Funções:** Definição, parâmetros, argumentos e retorno de valores.
*   **Módulos e Bibliotecas:** Uso de bibliotecas nativas como `math`, `random` e `datetime`.
*   **Manipulação de Arquivos:** Leitura e escrita de dados, com foco prático em arquivos **CSV**.

### 4. Tópicos Especiais
*   **Streamlit:** Desenvolvimento de dashboards interativos e aplicações web.
*   **APIs:** Consumo de dados e integração com serviços externos.
